What is Document Object Model (DOM) ?

The Document Object Model (DOM) is a programming interface for HTML and XML documents. It represents the structure of the document as a tree of objects, allowing you to manipulate its content and visual presentation. Each object represents a part of the document, such as an element or attribute.

How do you select on element from the DOM ?

The DOM Input select() method selects all the text content of a textarea or an input element that contains the text field.

What is event delegotion in the context of the DOM , and why is it usefull?

Event delegation is a design pattern in JavaScript used to efficiently manage and handle events on multiple child elements by attaching a single event listener to a common ancestor element.
